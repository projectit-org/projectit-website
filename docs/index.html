ProjectIt Language Workbench

This is the previous homepage of ProjectIt, a workbench for developing Domain Specific Languages (DSL) that works natively on the web.

Because the nature of ProjectIt has changed from a tool to create a projectional editor to a tool to create a full DSL, it was decided to change the name of the project to Freon.

This project has moved to Freon.
